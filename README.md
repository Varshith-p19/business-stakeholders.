Sales Performance Dashboard – Tableau Project

This project showcases an interactive **Sales Performance Dashboard** created using Tableau.  
It is designed for business stakeholders to quickly monitor key metrics like **Sales**, **Profit**, and **Growth** with clear visual insights.



Project Overview

The goal of this dashboard is to:

- Track and visualize **sales performance**
- Analyze **profitability** across different categories and regions
- Identify **trends over time**
- Provide quick insights using KPI cards
- Offer interactive filtering options for better decision-making

This project demonstrates my skills in data cleaning, visualization, dashboard design, and business storytelling.



Dataset Used

The dataset contains the following fields:

Order_ID
Order_Date
Region
Category
Product
Sales
Profit
Growth_Percentage

The dataset includes multiple years (2022–2024) to enable time-series analysis.



Key Features of the Dashboard

1. KPI Summary Cards**
- Total Sales  
- Total Profit  
- Average Growth  
- Total Orders  

These provide a quick business summary.

2. Time-Series Trends**
- Monthly Sales Trend  
- Profit Trend  
- Year-over-Year comparison  

3. Category & Region Analysis**
- Sales and Profit by Region  
- Category-level performance breakdown  
- Top 10 products by sales  

4. Interactive Filters
Users can filter the dashboard by:
- Region  
- Category  
- Year  
- Product  

This helps stakeholders explore insights dynamically.

5. Consistent Color Theme
A professional color scheme was applied:
- Blue → Sales  
- Green → Profit  
- Purple → Growth  
- Red → Loss  
- Soft neutral tones for categories
